# Session 15: Conditionals

## Introduction

A conditional sentence has an if clause and a result clause. It tells what happens if something else happens.

## Five Types

Zero conditional is for general truths:
If you heat water, it boils.
Pattern: if + present, present.

First conditional is for real future possibility:
If it rains, we will cancel the picnic.
Pattern: if + present, will + V1.

Second conditional is for unreal present or imagination:
If I had a million rupees, I would buy a farm.
Pattern: if + past, would + V1.

Third conditional is for unreal past or regret:
If I had studied harder, I would have passed.
Pattern: if + had + V3, would have + V3.

Mixed conditional connects past condition to present result:
If I had learnt French, I would speak it today.

## Common Mistakes

Wrong: If I will see him, I will tell him.
Correct: If I see him, I will tell him.

Do not use will in the if clause of first conditional.

Wrong: If I would have studied, I would have passed.
Correct: If I had studied, I would have passed.

Wrong: If I was rich, I would buy a car.
Formal: If I were rich, I would buy a car.

## Practice

1. If I ___ the Prime Minister, I ___ education free. (be/make)
2. Identify: If I had left earlier, I would have caught the flight.
3. Water freezes if temperature ___ below 0 degrees. (fall)
4. Correct: If I will get time, I will call you.

Answers:

1. were, would make
2. third conditional
3. falls
4. If I get time, I will call you.
